README – Talleres de Fundamentos en JavaScript
Aprendiz: Mariana Uribe

Objetivo
Desarrollar y reforzar habilidades en JavaScript puro mediante la práctica de manipulación de variables, métodos de arrays (map, filter, find, reduce), creación y manipulación de objetos, uso de funciones flecha, manejo de referencias e implementación de lógica tipo CRUD.
Proyectos Incluidos
1. Ejercicios Básicos
- Cálculo de promedio y validación de estado.
- Uso de filter() para obtener aprobados.
- Uso de reduce() para calcular promedio general.
- Uso de map() para extraer información.
- Creación de objetos dinámicos y desestructuración.
2. Mini Taller – Lista de Tareas
- Mostrar tareas.
- Agregar tareas.
- Marcar tareas como completadas (usando map e inmutabilidad).
- Eliminar tareas (usando filter).
- Buscar tareas (usando find).
- Obtener estadísticas de progreso.
3. Gestión de Aprendices, Productos y Contactos
- Filtrar aprobados.
- Calcular promedios.
- Aplicar descuentos.
- Ordenar productos.
- Agregar, buscar, actualizar y eliminar contactos.
- Exportar datos en formato JSON.
Aprendizajes Clave
- Comprensión profunda de métodos de arrays.
- Diferencia entre copiar valores y copiar referencias.
- Uso del spread operator para evitar mutaciones.
- Aplicación de lógica estructurada en proyectos pequeños.
- Implementación básica de estructuras tipo CRUD.
Reflexión Personal
Estos ejercicios fortalecieron mi lógica de programación y mi comprensión sobre cómo funciona JavaScript internamente, especialmente en el manejo de arrays y objetos. Además, permitieron aplicar conceptos de inmutabilidad y manipulación de datos fundamentales para proyectos reales.
